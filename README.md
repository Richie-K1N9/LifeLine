# Lifeline
Yr11 SDD Health Project

Features:
 - Setup Menu
    - Name
    - Gender (M/F/O)
    - Age
    - Height
    - Weight
    - Recommended Fitness plan
    - Other Fitness plans (easier and more difficult)
 - Main Menu
    - Goals
        - Fitness plan you chose in setup phase
        - Step Counter
        - Sleep Tracker
        - BMI
    - Exercises
        - Push Ups
        - Sit ups
        - HIIT 
        - 
    - Diet
        - Calorie tracker
        - Suggested food
        - recipes
