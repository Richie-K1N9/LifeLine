# LifeLine
Yr11 SDD Health Project

Features:
 - Setup Menu
    - Name
    - Age
    - Height
    - Weight
    - Recommended Fitness plan
    - Other Fitness plans (easier and more difficult)
 - Main Menu
    - Goals
        - Fitness plan you chose in setup phase
        - Step Counter
        - Sleep Tracker
        - BMI
        - 10000 Steps
        - 7-8 hr Sleep
    - Exercises
        - Push Up
        - Situp
        - HIIT
        - Other fat burning excercises
    - Diet
        - Calorie tracker
        - Recommended foods
        - Food Recipes
